Edirom User Forum

Organisation: Nikolaos Beer

Zeit: Mittwoch, 26. 9., 17:45 bis 19:00 Uhr
Raum: F0.225

Auf der diesjährigen EDIROM Summer School wird erstmalig das EDIROM User Forum stattfinden. In dessen Kontext ergibt sich für Editoren bzw. Editionsprojekte, die schon mit den EDIROM Tools arbeiten, die Möglichkeit, sich exemplarisch über projektspezifische Anwendungsszenarien auszutauschen. Gleichzeitig soll das Forum potentiellen Interessenten, die digitale Editionsprojekte planen, einen praktischen Einblick in die editorischen Möglichkeiten beim Einsatz der EDIROM Tools gewähren.

Die Veranstaltung beginnt mit ca. 10-minütigen Überblicksreferaten von Seiten der beteiligten Editionsprojekte, in denen die jeweils projektspezifischen Problemfelder erläutert werden. Dabei sollen neben den dort jeweils schon umgesetzten Lösungsansätzen auch Anforderungen zur Sprache kommen, die für die weitere Entwicklung der EDIROM Tools von Bedeutung sein werden. Die anschließende Diskussionsrunde wird diese Anforderungen in einem breiteren Kontext betrachten. Insbesondere soll hier Interessenten der Raum für konkrete Anwendungsfragen gegeben werden.

Ziel der Veranstaltung ist neben der Erstellung eines potentiellen Anforderungskatalogs für zukünftige Entwicklungen auch die Etablierung des User Forums über die Summer School hinaus sein.
