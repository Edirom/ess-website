An introduction to ODD

Dozenten: Raffaele Viglianti und Peter Stadler

Zeit: Freitag, 28. 9., 9:00 bis 17:30 Uhr
Raum: F1.406

This one-day workshop intends to give a fundamental overview on the TEI ODD meta language. ODD is both used to describe the formal specification of one's XML schema as well as the prose description of the according encoding practice (i.e. guidelines). While it can be edited through the Roma web interface but also by hand, both ways will be presented during the course of the workshop. This workshop will be held in English.

Prerequisites: Basic knowledge of the TEI, as imparted by the introductory workshop on TEI earlier that week.

Software Requirements: Browser (e.g. Firefox) and XML editor (e.g. oXygen)
