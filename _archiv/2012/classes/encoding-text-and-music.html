This seminar will introduce the encoding of textual sources containing music notation. It will discuss how music and text influence each other typographically and it will focus on the use of the TEI element notatedMusic in combination with MEI. This course will be held in English.

Prerequisites: Some basic understanding of TEI and MEI is strongly recommended.

Software Requirements: XML editor (e.g. oXygen).

This course will be held in english.
It is essential to bring an own computer!
